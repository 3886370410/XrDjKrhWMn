# 项目简介

## 核心功能
- 基于YOLOv8的电线杆部件缺陷检测系统，支持5类缺陷识别：Binding_irregularity、Parallel_clamp_protective_shell_missing、Tension_clamp_protective_shell_missing、Crossbar_corrosion、Tower_head_damage
- 支持图片、视频及摄像头三种方式进行检测
- 提供UI界面，实时显示目标位置、目标总数、置信度等信息
- 支持批量检测功能，可查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 可选择显示标签和原图
- 可选择检测模型

## 适用场景
- 电力设施巡检与维护
- 电线杆部件缺陷自动化检测
- 输电线路安全监测
- 电力设备状态评估
- 基础设施安全管理

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含10000张标注数据集
- 提供完整源码、UI界面源文件、图标、数据集图片和项目运行教程
- 已训练好模型，配置环境后可直接使用
